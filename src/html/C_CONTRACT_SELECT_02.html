계약 선택

계약 진행할 지역의 서비스명을 선택해주세요.
사업자 구분을 선택해 주세요.
가맹점 계약사항을 선택해주세요.
가맹점 계약 시 아래 서류들은 미리 준비해주세요.
  • 법인사업자 :  사업자등록증 사본, 법인등기부등본 사본, 실제소유자 확인서류
    (주주명부, 출자자/출연자 명부, 정관 등, 통장 사본)
  • 개인사업자 :  사업자등록증 사본
  • 비영리법인 :  설립목적검증서류(정관, 규약, 회칙 등)
  • 법정대리인 :  법정대리인신분증 사본, 가족관계증명서 사본
  • 대리인 :  대리인신분증 사본, 대리인 권한 확인서류( 위임장, 법인인감증명서 등)
